Conteúdo do Trabalho

O notebook contém as soluções computacionais e analíticas das seguintes questões:

Questão 1
- Plot do sinal contínuo usando degrau unitário
- Cálculo da energia do sinal:
  - forma analítica
  - forma numérica (aproximação por soma)
- Comparação entre os resultados

Questão 2
- Construção analítica do sinal da Figura 1
- Plot do sinal original
- Transformações no tempo:
  - \(x(-t)\)
  - \(x(t+4)\)
  - \(x(t-2)\)
  - \(x(2t)\)
  - \(x(t/2)\)

Questão 3
- Simulação de modulador AM
- Plot do sinal modulado e da envoltória

Questão 4
- Análise do sistema LTI:
  - causalidade
  - estabilidade
- Convolução:
  - analítica (expressão por partes)
  - numérica (aproximação discreta)
- Comparação gráfica entre analítico e numérico
- Repetição para:
  - \(h(t)=e^{-2t}u(t)\)
  - \(h(t)=e^{t}u(t)\)

Questão 5
- Solução de equação diferencial ordinária (EDO):
  - método clássico (analítico)
  - resolução simbólica com Sympy
- Plot do sinal de saída
